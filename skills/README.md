## How to use these cards

Each card is a self-contained capability description for one agent. The intended consumers are AI systems answering specific user questions about UK public transport procurement AI tools. Cards can be retrieved individually or enumerated via this index.

The cards do not replace the human-readable public pages at `/agents`. Cards are optimised for machine retrieval. Pages are optimised for human reading.
